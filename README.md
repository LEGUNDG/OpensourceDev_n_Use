# 오픈소스 개발과 활용
13~14주차 조별활동용 사이트

## 소개
[사이트](https://legundg.github.io/OpensourceDev_n_Use/)와 조원: 이정민(조장), 최운혁, 최찬영

## 개발 계획

**주제 선정** 
- 문화생활기록 아카이브
  -소주제: 연극, 영화, 뮤지컬

**홈페이지 레이아웃** 
- 메인화면
   - 상단 영역(Header)와 메인 콘텐츠 소개 영역, 글 목록 영역으로 나뉜다.
   - 메인 콘텐츠 영역은 크게 4분활로 좌측(1분활)에는 CULTURE LOG에 대한 소개가 배치되어 있고, 우측(3분활)에는 콘텐츠의 주요 카테고리(영화, 연극, 뮤지컬)를 나타내는 3개의 카드형 요소가 가로로 나란히 배치되어 있다.
   - 글 목록 영역은 좌측에 글목록 제목이 우측에 카테고리 필터링 영역이 배치되어있다.
- 상세 콘텐츠 페이지
   - 상단 영역(Header)와 콘텐츠 상세 정보 영역, 세부 내용 영역으로 나뉜다.
   - 콘텐츠 상세 정보 영역은 크게 2분활로 좌측(1분활)에는 작품 이미지가 배치되고, 우측(1분활)에는 작품의 간결한 설명이 배치되어있다.
   - 세부 내용 영역에는 작품 소개, 줄거리, 출연 배우, OST, 감상평, 메인화면 백 버튼이 가로로 배치되어있다.

**제작할 게시글의 역할 분담** 
- 영화(최운혁), 연극(이정민), 뮤지컬(최찬영)

**WBS와 간트차트** 
- 개발하면서 작성 예정
